# Machine Vision Experiments

本仓库为智能科学与技术23-3班孙浩泽（学号2023217595）的机器视觉课程实验项目。项目从传统图像处理出发，逐步过渡到基于深度学习的目标检测，涵盖完整的机器视觉技术链路。

---

## 📌 项目内容概览

### 实验一：图像滤波与特征提取
- 手动实现图像卷积（Sobel 算子与自定义卷积核）
- RGB / 灰度直方图统计分析（均值、熵等）
- 基于 LBP 的纹理特征提取与存储  
- 不依赖高级 API，强化对卷积与特征计算原理的理解

### 实验二：车道线检测
- 基于 Canny 边缘检测与 Hough 变换的车道线识别
- 引入 ROI 区域裁剪与几何约束抑制噪声
- 使用 HLS 颜色空间提升复杂光照下的鲁棒性
- 支持多车道与真实校园道路场景

### 实验三：手写数字识别
- 基于 MNIST 数据集训练 CNN 模型
- 结合传统图像处理完成真实场景数字分割
- 实现真实学号照片的端到端识别
- 体现传统视觉与深度学习的融合思想

### 实验四：目标检测（共享单车）
- 基于 SSD 框架的目标检测实现
- 对比 MobileNet 与 ResNet 等不同主干网络
- 使用迁移学习在小样本校园数据集上进行微调
- 多指标评估（mAP、IoU、PR 曲线、混淆矩阵）
